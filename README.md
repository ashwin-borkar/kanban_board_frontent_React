# 🚀 Kanban Board Frontend

A modern and responsive Kanban Board frontend application built with **React**, **Vite**, and **Tailwind CSS** featuring drag-and-drop task management and seamless integration with the Rails backend API.

---

# ✨ Features

- 📌 Create Tasks / Cards
- ✏️ Edit Tasks
- 🗑️ Delete Tasks
- 📂 Create and Manage Columns
- 🔄 Drag & Drop Cards
- 🕒 Historical Board View
- ⚡ Fast UI with Vite
- 🎨 Responsive Modern UI
- 🔗 REST API Integration

📦 Installation
1️⃣ Clone Repository
git clone git@github.com:ashwin-borkar/kanban_board_frontent_React.git
2️⃣ Navigate to Project
cd kanban_board_frontent_React
3️⃣ Install Dependencies

Using npm:

npm install

or using yarn:

yarn install
▶️ Run Frontend Project

Start Vite development server:

npm run dev

Frontend runs on:

http://localhost:5173
🔥 Run Backend Server

Make sure Rails backend is running.

Navigate to backend project:

cd kanban_board

Start Rails server:

bin/rails server

Backend API runs on:

http://localhost:3000
⚙️ Environment Variables

Create .env file:

touch .env

Add:

VITE_API_BASE_URL=http://localhost:3000/api/v1
📡 API Endpoints
Method	Endpoint	Description
GET	/api/v1/columns	Fetch Columns
POST	/api/v1/columns	Create Column
PATCH	/api/v1/columns/:id	Update Column
DELETE	/api/v1/columns/:id	Delete Column
GET	/api/v1/cards	Fetch Cards
POST	/api/v1/cards	Create Card
PATCH	/api/v1/cards/:id	Update Card
DELETE	/api/v1/cards/:id	Delete Card
